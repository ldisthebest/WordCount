exe使用方法：
按住win+R键打开“运行”窗口，先输入exe文件的绝对地址（绝对地址查看方式：右键点击exe文件，点击“属性”，点击“安全”，“对象名称”后的即为此文件的绝对路径，可直接复制）；然后输入空格，然后便可输入作业要求命令，输入完毕点击“确定”即可执行，如出现弹框，点击“ok”即可，然后打开输出文件便可看到结果。
如果需要读取命令中指定文件，请在exe当前目录下自行创建，否则文件找不到则报错。
注意：
1.当用-s命令时，要输入绝对路径，如F:\softwareQuality&Test\WordCount\bin\*.c
2.src文件中为源代码，可通过文本方式打开查看
3.由于myeclipse30天试用后付费原因，本人调整了本机系统时间以骗过myeclipse到期检测，所以在git上提交的时间可能会很奇怪。
